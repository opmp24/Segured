# Demo PWA — Seguridad Laboral

Proyecto demostrativo: Progressive Web App simple y elegante para mostrar a un cliente.

Archivos principales:
- index.html
- styles.css
- app.js
- sw.js
- manifest.json
- icons/

Probar localmente (puerto 5000):

```bash
# Con Python 3
python -m http.server 5000

# ó con npx serve
npx serve -p 5000
```

Abrir http://localhost:5000 en un navegador moderno. Para probar la instalación, use Chrome/Edge en modo HTTPS o `localhost` (sirve en local sin HTTPS).
